Template for a basic Node application that has Typescript, ESLint and Prettier.

Install modules:

```shell
npm install
```

Start development server:

```shell
npm run dev
```

Build project:

```shell
npm run build
```

Run the build:

```shell
npm run start
```

Lint the project:

```shell
npm run lint
```

Reformat the project:

```shell
npm run format
```
